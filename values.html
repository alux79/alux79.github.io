Traditional & Core Values

Importance of Traditional Values

How much should tradition affect education in a quickly modernizing world? Forming an understanding of the role of traditional values in both the past and present is necessary to ponder the complexity of current educational reforms. For a while, Cambodian education has been guided by values like collectivism, respect for authority, social hierarchy, risk avoidance (Berkvens, 2017). These values are deeply rooted in Cambodia’s heritage, and have greatly defined the educational environment of Cambodia. However, these ideas conflict and take pressure from modernized and individualistic learning models (Heng & Koemhong, 2022). This brings the question, are traditional values still effective in today’s environment? Can these values be evolved to better fit the educational landscape?

Adapting Traditional Values

Looking at these values closely, we see none of them being entirely beneficial or inherently harmful. Much of the influence that these values hold depend on context and how they’re implemented. The values like respect for authority and risk avoidance definitely facilitate stability, accountability, and community, which are no doubt important in both personal and academic growth (Purwaningsih & Ridha, 2024; Chaika, 2021). Nevertheless, these same traditions can harm the same growth and progress by discouraging critical thinking, individual expression, and innovation (Sen 2022). This is where the real question is, not determining if these values belong, but rather how can they be meaningfully adapted. Ideally, respect can exist without suppressing inquiry, creativity should be encouraged, but that doesn’t mean we won’t have stability, many values should be able to coexist.
